# Prototype Privacy Notice

This experience is an early prototype of the Grand Pannonia Hotel mystery game.

## Read This First

- Do **not** enter any sensitive, personal, financial, medical, legal, account, or confidential information.
- Treat all chat messages as prototype gameplay input only.
- If something is private in real life, do not type it here.

## Where Your Data Goes

When you use the prototype, your gameplay data can flow through:

1. The Railway-hosted backend service that runs this application.
2. OpenAI services used to power the in-game agents and room/image changes.

## What We Store In This Prototype

This prototype stores only the minimum state needed to run one play session, including:

- an anonymous session identifier in a browser cookie
- current room state and room-image history for that anonymous session
- current suitcase inventory for that anonymous session
- OpenAI conversation identifiers used to continue the same gameplay thread

The prototype is designed to avoid storing full chat transcripts in the local project database whenever OpenAI-hosted conversation state can be used instead.

## What You Should Assume

- This is not a secure personal messaging system.
- This is not a place to upload or share secrets.
- Prototype behavior may change, fail, or reset without notice.
- Generated responses and generated room edits may be imperfect.

## Consent

By starting the prototype, you confirm that:

- you understand this is a prototype
- you will not submit sensitive data
- you understand that gameplay data may be processed through Railway and OpenAI services

## Current Scope

This notice is a plain-language prototype warning and not a full production privacy policy.
