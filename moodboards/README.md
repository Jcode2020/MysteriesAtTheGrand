# Grand Hotel Mood Board

This folder contains the single canonical visual reference for the Grand Hotel mystery experience.

## Source Of Truth

- `_merge_final.html`: the project's approved moodboard and design-language reference.

## How to use it

1. Open `_merge_final.html` in a browser.
2. Use it as the source of truth for palette, typography, layout tone, and interface styling.
3. When a design choice is unclear, prefer the option that stays closest to this board.

## Why this board

- It preserves the warm hotel palette and scenic luxury needed for the project vision.
- It adds the dossier-style presentation system that gives the experience a stronger investigative identity.
- It is the only moodboard that should guide future visual decisions unless a human explicitly replaces it.
